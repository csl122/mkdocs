# Starship

tags::  Terminal, Tool, Theme

- Installation #安装
	- ```
	  brew tap homebrew/cask-fonts
	  brew install --cask font-caskaydia-cove-nerd-font
	  设置好字体
	  brew install starship
	  在.zshrc中加入init指令
	  eval "$(starship init zsh)"
	  mkdir -p ~/.config && touch ~/.config/starship.toml
	  将下面的文件放入该目录
	  ```
